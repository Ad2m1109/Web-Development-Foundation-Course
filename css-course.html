CSS Reference Guide

Selectors

Learn how to select HTML elements to style them with CSS.

Basic Selectors

/* CSS */
p {
  color: blue;
}
#main-title {
  font-size: 2rem;
}
.highlight {
  background: yellow;
}

Practice Exercise

Try changing the color of the paragraph and the background of the highlighted text.

Hint: The default HTML contains:
  • <h1 id="main-title">Hello CSS!</h1> (ID: main-title)
  • <p>This is a <span class="highlight">highlighted</span> paragraph.</p> (Class: highlight)

Layout

Arrange elements on the page using Flexbox and Grid.

Flexbox Layout

/* CSS */
.flex-container {
  display: flex;
  gap: 10px;
}
.flex-item {
  background: #c8e6c9;
  padding: 10px;
  border-radius: 4px;
}

Practice Exercise

Use Flexbox to arrange the items in a row with space between them.

Hint: The default HTML contains:
  • <div class="flex-container">...</div> (Class: flex-container)
  • Inside: <div class="flex-item">Item 1</div>, Item 2, Item 3

Grid Layout

/* CSS */
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}
.grid-item {
  background: #ffe082;
  padding: 10px;
  border-radius: 4px;
}

Practice Exercise

Use CSS Grid to arrange the items in three columns.

Hint: The default HTML contains:
  • <div class="grid-container">...</div> (Class: grid-container)
  • Inside: <div class="grid-item">A</div>, B, C

Responsive Design

Make your web pages look good on all devices using media queries.

Media Queries

/* CSS */
.box {
  width: 300px;
  background: #b3e5fc;
}
@media (max-width: 600px) {
  .box {
    width: 100%;
    background: #ffccbc;
  }
}

Practice Exercise

Make the box full width and change its color on small screens.

Hint: The default HTML contains:
  • <div class="box">Resize the window!</div> (Class: box)

Transitions & Animations

Add smooth effects and animations to your elements using CSS transitions and keyframes.

Transitions

/* CSS */
.button {
  background: #4CAF50;
  color: white;
  padding: 10px 24px;
  border: none;
  border-radius: 4px;
  transition: background 0.3s;
}
.button:hover {
  background: #388e3c;
}

Practice Exercise

Add a transition effect to the button's background color when hovered.

Hint: The default HTML contains:
  • <button class="button">Hover me!</button> (Class: button)

Animations

/* CSS */
@keyframes moveRight {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}
.animated-box {
  width: 60px;
  height: 60px;
  background: #ff9800;
  border-radius: 8px;
  animation: moveRight 1s linear infinite alternate;
}

Practice Exercise

Animate the box to move right and left using keyframes.

Hint: The default HTML contains:
  • <div class="animated-box"></div> (Class: animated-box)
